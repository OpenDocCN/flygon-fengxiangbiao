# 俄罗斯电商 Wildberries 即将招商中国卖家，体量是 OZON 的两倍，适合跨境销售

> 原文：[`www.yuque.com/for_lazy/xkrm14/tfvywur66upkkgl0`](https://www.yuque.com/for_lazy/xkrm14/tfvywur66upkkgl0)

正文：

俄罗斯电商 Wildberries（）正在内测白名单，即将要对中国卖家进行招商，根据财报来看体量是另一个俄罗斯本土电商 OZON 的两倍。平台调性快时尚为主类似 SHEIN，非标居多适合跨境，相较于 OZON 来说比较难跟卖，另外物流方面也有一定的优势。做跨境的圈友们可以提前针对性看一下市场提早布局。[`www.wildberries.ru/`](https://www.wildberries.ru/)
