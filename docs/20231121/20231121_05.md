# 利用视频号和小黄车快速出单的方法

> 原文：[`www.yuque.com/for_lazy/xkrm14/geugluwh5fb8nl0t`](https://www.yuque.com/for_lazy/xkrm14/geugluwh5fb8nl0t)

正文：

视频号-数字人书单号（每天半小时，快速出单） 朋友第一条和第二条挂小黄车的均出单了。 朋友测试成功的一个项目，大概执行步骤：
1、确定要卖某个名人的书籍，比如巴菲特，可以选择一些职场、情感、智慧、成长赛道国外知名人物的书籍，内容优质的（或者找视频号选品里面非常爆款的书籍）
2、从抖音上或者易撰等找一些爆款文案 3、利用 HeyGen 来生成短视频：一般一分钟以内即可 4、开橱窗，视频挂载小黄车
前期冷启动的时候没办法开橱窗，有可能需要一个月左右的周期，这个时候可以选择用其他的方式把粉丝加上，然后直接开橱窗，节省时间成本。
Tips：挂车的时候有新的规定，不要去用箭头、手指之类的引导点击小黄车，会违规。
