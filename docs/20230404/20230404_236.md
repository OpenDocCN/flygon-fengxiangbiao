朋友圈看到有人做 WiFi 扫码变现。 现在有一种技术把连 WiFi 的密码过程变成扫二维码，用户在扫码连 WiFi 的过程中会有一段广告时间，每扫一次，商家和推广人都可以获得分成。 需要自己注册公司，成为代理，然后全国招商，不断推广。
