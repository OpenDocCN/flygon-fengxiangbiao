# 半天妖烤鱼通过发传单引流进店，为何选择这种方式？

> 原文：[`www.yuque.com/for_lazy/xkrm14/ish4f9feyognk1pb`](https://www.yuque.com/for_lazy/xkrm14/ish4f9feyognk1pb)



作者： 酱图森



日期：2023-08-07



点赞数：55



正文：



发疯了，半天妖疯了一样发传单。 目前有 1400 家门店的半天妖烤鱼，还在通过发传单的方式引流进店。 其河南分公司一天爬楼上百层，到户发了 1106 张，挂车分发了 694 辆。 有如此规模的企业，为什么还在用看似傻乎乎的方式推广呢？ 有人统计过，传单转化率通常在在 0.5%到 1%左右。也就是说，你发 1000 张印着降价优惠的传单，大概会有 5-10 个人进店。 但如果掌握特殊的技巧，这个概率可以大幅度提升，我见过最高转化的案例有 23%之多。 小马宋给甜啦啦做过发传单实验。员工一小时发了 33 张，吉祥物熊黑蛋一小时发了 83 张，是员工的 2.5 倍。 所以你看，发传单是非常有技巧的一项工作。什么人发，发给什么人，传单设计的内容、时间、地点，用什么样的话术都需要充分考虑，才能真正让传单起到作用。 甚至要求高的，一人一策，一址一策，同一家店铺，分发不同人群的传单都会有略微区别。我们曾经服务的一个客户，光迭代优化的传单方案就有 3 版之多。 但无论如何其核心的思考方式，就是站在顾客视角去看，他们如何发现，如何决策，如何吸引。 毕竟发传单并不是推广目地，吸引顾客到店才是最主要的目地。只有想办法把顾客招呼进来了，才会有后续的可能嘛。 最简单的手段往往也是最有效的。不管怎么样，老板们传单发起来吧！
