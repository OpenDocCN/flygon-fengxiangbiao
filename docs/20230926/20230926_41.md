# 新疆和田的种地政策和商机分析

正文：

通过来新疆和田这几天，谈谈自己的想法 国之根本还是农业，农业离不开种地，今天就来谈谈新疆和田这边的种地政策，
我在的这个地方和田下属的一个县城，目前种花生是补贴 500/亩，其他的不太清楚，反正这边有种葡萄，西瓜，红枣，玉米，板蓝根的， 然后我说下这边的商机，
商机 1:自己承包地（一般承包地都是签约 1-3 年的合同)可以种植，板蓝根，玉米，红枣，花生，葡萄，西瓜等，这种包地成本在 150-500/亩，这种好处就是投入成本低基本上 10-20 个就可以做了，做的好一年不仅能回本还可以盈利，属于风险小。
商机 2:开荒地，就是把从未开发的土地，自己投入挖掘机，人工，进行平整，基本上一亩地投入在 800-1000 左右，如果你要开荒地，需要 5000 亩起，属于投入成本比较大，但是也有好处，开荒地，不需要承包地的成本，可以直接签约 5-10 年的合同，甚至 20 年的合同，就是你荒地开出来，剩下的时间不管你是自己种植，还是承包都不会亏，这个需要投入成本很大，没有一定资金的不建议搞
商机 3:因为新疆这边每天的温差比较大，种植出来的水果（哈密的哈密瓜瓜，库车的小白杏，吐鲁番的葡萄，阿克苏的红富士苹果）都是全国有名的，还有各种农副产品都适合在新疆这边土壤生长，尤其是新疆的棉花，这个我就不多说了，如果有直播团队对于这块感兴趣的，也可以过来考察，直接在地里直播，符合人货场的条件
