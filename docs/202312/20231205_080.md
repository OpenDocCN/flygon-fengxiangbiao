# 疫情后名校游学受限，高校资源成为家长群聊天的商机

> 原文：[`www.yuque.com/for_lazy/xkrm14/xdwlhqaridpcarnp`](https://www.yuque.com/for_lazy/xkrm14/xdwlhqaridpcarnp)

作者： 远方

日期：2023-12-01

点赞数：**60**

* * *

正文：

家长群聊天看到的商机。宝妈们带孩子进名校，感受有名高校的氛围，这样的励志游一直有需求。但疫情后很多名校再也不能随便进了。这给有高校资源或在高校的学生带来了机会，如例子中提到的研究生带下家长一家日入一千，还是很香的。毕竟报游学机构不反旺季难约，且人数有限还很贵。寒假马上要开始了，有资源的小伙伴搞起来。
