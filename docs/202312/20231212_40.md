# 注册商标的重要性，否则可能遭遇品牌侵权

> 原文：[`www.yuque.com/for_lazy/xkrm14/gmlcnzeo9y2go1aq`](https://www.yuque.com/for_lazy/xkrm14/gmlcnzeo9y2go1aq)

正文：

一个提醒，大家的个人 IP 名称、产品名称都请尽早注册商标，否则会让你遗憾一辈子。
你注册了一个公众号，取了一个独特的品牌名称，经过一段时间的运营，取得了不错的名气，也有挺多粉丝关注。
然后有个人用你的品牌名称去注册商标，向微信投诉你侵权了对方的商标，微信就会把你的名字清掉。
这个时候，你再去申诉，说明你用这个品牌名称在先，对方恶意注册在后，很有可能能够申诉成功，微信会给你恢复名字。 但同样的事情，如果换成 App
名称，被别人抢走商标之后，应用市场不会管你是否是先注册先使用，只认商标。
于是你的 App 名字就会被对方抢走，对方再上线一个同样功能同样名字的 App 来抢你的用户。 你只能忍痛改名字，从头再来。
