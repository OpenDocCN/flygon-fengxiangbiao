# 陪护行业赚钱机会大，关键是搞定流量端和接单派单

正文：

因为自己近期生病手术，了解了下陪护。 1.陪护这个行业，陪护都是接单赚钱，有点类似于家政。一单 300，陪护阿姨到手 250 左右。
2.看了下小红书上的账号，粉丝量不高，但评论区都是问价的。 3、陪护细节区分，报价是否管饭。具体的工作清单。 4、陪护除了接单 还能卖点用品
比如一些病人无法自理的床上便盆，产褥垫，临时要价格都比较高。 5.搞定流量端，加入陪护阿姨群，接单派单就行。

评论区：

梁有鱼🐟 : 收藏了，直接对标做[偷笑]

婧雯 : 这个前提是他已经进了医院，现在的医院陪护机构都特别的稳定，你没得关系根本进不去。里面的陪护都是机构自动排单，根本搞不赢接你外面的单子。除非你直接和机构对接流量。

颜乐乐 : 医院有自己的陪护机构，但我在医院也看到了外面其他家政的陪护。陪护阿姨也有自己的接单渠道。我自己和请的陪护聊下来，他们单也不是很饱和，每个月出单 20 来天。不出单的日子是没有收入的，机构也没有给他们交社保。
