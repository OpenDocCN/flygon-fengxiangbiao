# 个性化朋友圈欢迎语，给每个好友写一条专属问候

> 原文：[`www.yuque.com/for_lazy/xkrm14/sx6cb8x1tibcvnpz`](https://www.yuque.com/for_lazy/xkrm14/sx6cb8x1tibcvnpz)

作者： 哥飞

日期：2023-08-09

点赞数：84

正文：

朋友圈置顶 哥飞想到一个好玩的玩法，给每个好友写一条“XXX 你好，欢迎来逛我朋友圈”。 XXX 要替换为好友名义，然后设置分组可见，实际这一条仅这个好友可见。 最后的效果就是，好友们点进去，各自能够看到自己名字的欢迎语。 目前不知道最多可以置顶多少条，但我测试 23 条都没问题，那么估计至少能置顶 30 条，当然这个数字也许可能会更大。
