正文：

分享一则来自美国那旮旯的生财故事，或许对你找到风向标并顺势而为能有所帮助。 昨天，《可乐周报》的创始人 Happy
Xiao 分享了一位成功转型投资人的独立开发者的故事，从他的故事中，我们就能发现如何在起风时发现更多的风向标。 Syed
Balkhi 是一位成功转型投资人的独立开发者，今年 32 岁的他，12 岁才从巴基斯坦移民美国。刚到美国时他不会说英语，为了在图书馆打游戏，自学 Proxy 代码，后来又帮人做 PHP 网站。
在 Wordpress 出现后，他专注做 Wpbeginner 网站（你一定听说过，因为它的 seo 屌爆）。之后不断收购 WordPress 插件，现在年入数千万刀，又买了十个加油站（守住财富制造现金流）。
从 Syed 的故事中，Happy Xiao 给出的建议是“找到下一个统治级的东西，做它的周边，与它一起成长（比如销售 Notion、打包 AI 程序等）”。
我个人认为，如果无法参与大掘金运动，不妨看看掘金者出了铲子以外还有哪些普遍存在的需求，满足他们的需求，你就能找到生财密码。
