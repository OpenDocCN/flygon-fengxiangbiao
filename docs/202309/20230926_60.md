# 九月考研季，如何准备考研项目？

> 原文：[`www.yuque.com/for_lazy/xkrm14/wy0729evenk8ttvg`](https://www.yuque.com/for_lazy/xkrm14/wy0729evenk8ttvg)

作者： 孙源梓 Log

日期：2023-09-18

点赞数：**67**

* * *

正文：

九月份到来，做考研相关项目的，可以着手做准备了。 1.学校陆续公布招生简章，可以做招生简章的解读课程以及往年录取情况分析
2.搜集相关专业的复试真题以及资料，初试结束以后会迎来一波很大的流量，考研的同学对复试真题需求量很大，也愿意花钱，毕竟都走到复试阶段了。
3.征集相关专业的学长学姐，推出复试相关课程。
4.可以用 QQ 群来承接流量，因为每个学校招生人数通常在 1000 人以上，报考人数在 5000 左右，QQ 群在人数和文件管理方面有优势
5.可以先从一个学校做起，甚至一个专业做起，吃透一个学校每年 30 万不是问题。
6.不要以赚钱为目的，以帮助他人上岸的心态来做，你才能长久，也才能建立信任，大家信任你，你自然会有钱可赚。钱是交易的媒介，它本质上是对“信用”的度量。
