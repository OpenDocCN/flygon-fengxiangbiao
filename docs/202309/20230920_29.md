# YouTube 账号用 155 条视频吸引 60 多万粉丝，教你如何推广海外市场产品

> 原文：[`www.yuque.com/for_lazy/xkrm14/pordzufhyh3qm9f2`](https://www.yuque.com/for_lazy/xkrm14/pordzufhyh3qm9f2)

正文：

如果你手头有好的产品想卖到海外市场，却不知道如何推广，不妨看看这个 YouTube 账号找找灵感。
这是一个今年 3 月在越南注册的账号，半年多时间，他们用 155 条视频吸引到了 60 多万的粉丝，创下了将近 3100 万的播放量。
至于这个账号为什么流量会爆，其实理由也很简单。因为他们的视频没有任何台词，就是工作人员演示自家产品的使用流程。如图二这款可拆卸的沙发，工作人员也是一气呵成地表演了组装的过程，整个画面很有趣也很有创意，更关键是证明了这款沙发的实用性。
当然了，目前这个账号貌似只想乖乖赚取流量收益，否则我也没有个人简介或者视频评论区看到外挂的链接。不过对于在做外贸、跨境电商的圈友们来说，这个账号背后的一套玩法，同样是值得参考的。
