# 电话招生成为中专学校的赚钱风向标

> 原文：[`www.yuque.com/for_lazy/xkrm14/km8hmq1ghz24a9gs`](https://www.yuque.com/for_lazy/xkrm14/km8hmq1ghz24a9gs)

正文：

给中专学校电话招生。
刚和隔壁公司的业务员聊天，过去两个月，他放着自己的主业没做，打了两个月的电话帮学校招生，小赚了大几万。据他说，一个学生大概有 1500-2000 的利润（有些专业更高），做的好的，有收入十几万的。经验是：1、能有几个人的小团队一起做，这样赚钱效率高，学校也更愿意相信团队的实力；2、多找几个学校合作，这样方便学生选择，提高成交率；3、要找教育局或者学校老师的资源，拿得到家长信息才能打电话；4、有人也在抖音上招生，如果会运营懂投放的，这也是获取线索比较高效的一种方式。
