# 短剧变现新思路：流量变粉丝，引流变现，多平台收益

> 原文：[`www.yuque.com/for_lazy/xkrm14/dun3cd9mvxohc6ox`](https://www.yuque.com/for_lazy/xkrm14/dun3cd9mvxohc6ox)

正文：

短剧在小红书变现也是一种新思路，流量大也能接一些小红书商单，但我不建议去接小红书商单，还不如多做一些短剧流量，多涨一些粉丝，然后再引流进粉丝群，再引流到微信，可以收徒变现，一单大几百的，也可以直接卖素材，一部短剧看完最起码要几十块钱，我在私域直接卖个 9.9、19.9 也是有不少人买的，也不要小看这个 9.9 这些低客单价，只要量上来了收益还是挺好的，我们还可以用来放进夸克网盘进行拉新变现，这样二次变现收益还是挺稳的，而且我们不止卖一部短剧的素材，可以卖多部短剧的素材一起打包卖，再加上还有抖音、快手、视频号的 cps 收益，每天都有不错的收益，也可以在这些平台引流私域，变现方法就非常清楚明了，大家感兴趣的可以去试试
