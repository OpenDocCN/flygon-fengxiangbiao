# 小红书可蹭”狂飙“热度引流

正文：

小红书上利用开年热播剧"狂飙"的一些经典画面，配上一些搞笑梗，做成电脑、手机、平板的壁纸和屏保，比如高启强向泰叔下跪拜干爹的那个名场面，配上文案："只要赚点窝囊费，当牛做马无所谓"，做成电脑桌面背景，在评论区一片求原图的留言，不管是引流还是变现，这效果杠杠的。
