# 用收钱的方式招人干活，解放时间又增加收益

正文：

最近了解到一个新的商业模式，感觉挺牛的。既可以放大自己的收益，又可以解放自己的时间。
很多公司花钱招人干活，担心员工干活动力不足，还要花精力去给他们做心理按摩，解决情绪上的问题。
如果转变一下模式，用收钱的形式招人干活，那是不是可以筛选出一批赚钱欲望比较强，执行力更足的员工？ 最近就有不少公司这么干。
先自己跑通一个网赚项目，然后到县城找大量的宝妈，不给底薪，反而收学费（一般两三百）。
主要就教怎么剪辑和直播，学些基本操作后直接开干，获得的利润按约定分成。不需要告诉什么底层逻辑，直接让她们按照 SOP 操作就行。
这个模式下，自己的项目可以很好的放大，也解决了管理上的问题，并且给宝妈们提供了一个赚钱机会。
这种模式的前提是，本人要有选品和测试爆款素材的能力，以及能有接触类似宝妈这类人群的渠道。
