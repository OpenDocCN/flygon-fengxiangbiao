# 使用地图需注意边界正确，否则可能面临高额罚款

> 原文：[`www.yuque.com/for_lazy/xkrm14/vvk246trhpndgx5s`](https://www.yuque.com/for_lazy/xkrm14/vvk246trhpndgx5s)

作者： 张云金 _GISer

日期：2023-10-23

点赞数：**50**

* * *

正文：

公开使用地图切记要注意边界正确，无论是公司网站还是印刷材料中，地图一定要确保完整，否则可能被处以数万元乃是十几万的罚款。 [行政处罚结果信息公开](https://www.zjzwfw.gov.cn/zjservice/matter/punishment/detail.do?unid=330102022023010001)
