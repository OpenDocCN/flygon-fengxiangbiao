正文：

GPTs 已经开放给所有 ChatGPT Plus 用户，人人都能不需要编程就创建 GPTs 。 GPTs
有很多使用场景，但我建议大家先去做最简单最容易做的场景，就是 Chat with files 。 创建一个 GPTs ，上传一份 markdown
文档，即可制作一个基于这个文档来回答问题的智能机器人。 那么你是否可以把所有开源项目的说明文档都传上去呢？ 一个项目一个 GPTs ，然后提交到一些第三方的
GPTs Store ，等待用户使用。 图中是一个例子。
