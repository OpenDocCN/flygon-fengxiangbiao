# 公司点餐群：省时省钱，送货上楼，利润可观

正文：

做餐饮、小吃、水果的可以试试去做公司点餐群今天看了下微信，我们公司已经有四五个点餐群了，生意还可以。
需求：每天点外卖需要花费不少时间，并且需要下楼去拿外卖。通过点单群点的话价格相对便宜，多了一个选择，并且会送货到所在楼层。
利润空间：我们公司这个工区三层楼，五六百人，四个群一起看下来有一百多单。四家平均都有二三十单。
个人认为如果是做点餐群的话，需要两三天更换一次菜品，只要不是特别难吃，每天还是有不少点单的。还有一点就是最好是通过找到一个公司员工来建群，然后这样他们传播起来会快一些。我们这几个群都是公司同事建的，然后公司内部自己去宣传。这个思路可以扩展到其他食品类目。
