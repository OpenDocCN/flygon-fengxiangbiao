# 抖音平台政策调整：重复发布限流封号，图文标题要长

> 原文：[`www.yuque.com/for_lazy/xkrm14/ousn9vl9ldigogig`](https://www.yuque.com/for_lazy/xkrm14/ousn9vl9ldigogig)

点赞数：**98**

正文：

做抖音的友情提醒： 11.8 日官方会议中的一些平台政策倾向调整整理：
1、标题、图片高度相似的内容重复发布；单个图文内容中，发布多张重复图片；一个月以内平台会针对这部分账号限流、封号
（图文赛道近期会重点打击同质化问题。相似标题文案&相同拍摄背景&相机拍摄角度） 2、图片分辨率不低于 1280*720
3、图文需要三张及以上，两张后面会开始限流 4、图文的标题，要长；要上产品相关干货、上消费场景铺垫、上功能悬念反问。 短标题会开始影响流量。
ps：正常娱乐视频的标题风格后期后受到打压，如，爱困不住我，我没有心，我自由到底#美出高级感（这里透露出抖音官方的挂车和娱乐的流量池已经区分开了，并且在区别制定流量池算法规则）
5、背景音乐需要和商品内容符合。这里权重不高，锦上添花的东西。
6、图片上不要贴价格，官方的流量考核规则里，贴了=画蛇添足，要贴最好只贴一个价格并且在产品焦点展示的图上。
7、打压擦边流量出单。如利用美女+好汉歌的反差感博流量出单。（核心在于平台对于电商侧生态定位还在于产品自身的卖点展示，并不想放开低俗/擦边的流量转化。）
