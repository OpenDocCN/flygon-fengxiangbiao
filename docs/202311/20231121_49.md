# 旧衣回收赛道在小红书、抖音等平台流量不错，加盟商机会多

正文：

旧衣回收这个赛道在小红书，抖音里面流量都不错，我有亲戚是做分拣厂的，之前成立了新媒体部门，目前在江浙沪地区已经招了 100 多家加盟商，有 5 个 5000 多平米的厂房，目前还在扩建中。
旧衣回收这一块水比较深，对于加盟商来说，他们一般对 c 端用户都是以统货 0.4-0.6 元/斤的价格上门回收旧衣，折算下来是 1000 元/吨，分拣厂给到他们的价格是统货 2000-2500 元/吨的价格，做的好的一天可以收 3 吨左右。另一部分就是加盟商再招代理，小红书想要招商加盟的人特别多，我们自己做了个招商号基本引来了 1000 多个客资，所以有很多加盟商打着源头分拣厂的旗号搞起了加盟。
对分拣厂来说，我们基本都是卖到越南，缅甸，非洲区域。收来的衣服到厂了还需要二次分拣，也有只收分拣好的服装鞋子直接出口，夏装跟鞋子类的卖的价格都会高不少
