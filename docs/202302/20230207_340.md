# 快团团快速转化方法：根据客户画像买一个符合条件的公众号，改名为快团 XXX，每周推送一条低价团购信息

快团团快速转化的一个方法： 根据客户画像，buy 一个符合条件的公众号（服务号），改名为快团 XXX，每周推送一条美妆低价团购信息。 1.可以吃搜索流量。 2.直接触达用户，点进去就是跟团信息，容易产生从众购买行为。 3.订阅团长后，以后每次开团，用户都会收到公号消息。
