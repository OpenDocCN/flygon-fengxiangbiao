# 潮汕人在深圳开柜台做电子生意，意外发现卖家乡美食更赚钱

点赞数：63

正文：

在外的人都经常想念家乡的美食，认识一位潮汕人原来在深圳开柜台做电子生意，偶尔从老家卖老家食品过深圳，结果越做单越多，副业变主业了，基本流程就是老家有固定的货源，搜集客户订单，然后寄放到客车大巴过来深圳，每天一趟。发散开，在别的城市一样，如果有类似的资源和渠道都可以开展，毕竟民以食为天和谁都有一个家乡胃。
