# 地推洗车店团队伙伴两个月躺赚过万，利用 ETC 和 POS 机合作实现三方共赢

正文：



团队伙伴，地推洗车店二个月，现月躺赚过万 和洗车店聊合作，第一个做的是 ETC，ETC 装一个提成大概在 130 这样，给老板打横幅在本店办理 ETC 免费送 3 次洗车。 让老板在自己朋友圈或客户群宣传在他的店里安装 ETC 送 3 次洗车。客户安装完后发海报照片发送到朋友圈，可以给门店做到拉新，送的洗车一个月只能用一次，3 个月有心的老板可以重新销售上套餐转为老用户。 洗车店安装一个以 130 为例我们收入是 130-90（3 次洗车）=40 元利润 第一个品起量后，马上加了第二个品 pos 机，做的是高提成的品，客户装机器需要二百押金，首刷一万后提成 400 元，这个可操作空间大，押金我们自己出，只需要客户刷一万元，送客户 4 次洗车，一个月用一次。 利润 400-200（我们出押金）-120（4 次洗车）=80 元 三方共赢，用户能额外得到实惠，店铺老板增加营业额并免费拉新客户，对于我们来说，薄利走量。
