# 时间相关的内容成为爆款，触发回忆的新方向

> 原文：[`www.yuque.com/for_lazy/xkrm14/zdbex2u9c6tdsune`](https://www.yuque.com/for_lazy/xkrm14/zdbex2u9c6tdsune)



作者： 进阶的阿讯

点赞数：62



正文：



这个博主前面的视频都播放量评论数都很平淡，但是最近出了去采访十年未见的同学的系列视频后每一期都成爆款了。 好像跟时间相关的内容都比较容易成为爆款，不管是生活 vlog、电视剧、电影。 每个人都只能知道自己的生活，很多人都想要看看自己以外的人，特别是以前与自己相关的人，后来过得怎么样，有没有哪些选择改变了他们。 和时间联系起来，去做内容，触发大家的回忆，可能是一个不错的方向
