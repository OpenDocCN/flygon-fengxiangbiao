# 宠物信息中介平台，帮助领养和出售宠物的交流

正文：

宠物信息发布平台 开设多个不同城市的微信号，朋友圈发布各种猫咪，狗狗领养或者出售的信息，不是自己卖或者发布领养，主要是做个信息中介平台，帮有要发布的人发布消息，发布信息不用钱，想要领养或者购买的买家需要支付信息费 100 元！ 可以和各种宠物医院合作，帮他们免费发布信息，然后通过小红书，闲鱼，微博等渠道发布领养信息，吸引需要的买家 这个需求很大，自己的猫咪就是通过这个渠道同城购买，同时也在上面发布过流浪猫领养，也很快领养出去了， 作为一个信息中介，让领养代替购买，又能赚到钱，有时间的可以搞起来！
