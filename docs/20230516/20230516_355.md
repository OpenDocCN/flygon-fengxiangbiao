# 财富榜就是信息榜就是机会榜

<ne-text id="uf79e778a">正文：</ne-text>



<ne-text id="u393aadbb">财富榜就是信息榜就是机会榜。</ne-text> <ne-text id="uff613904">谁当世界首富，跟我有什么关系呢？你别说，还真有点关系。因为富豪榜，从某种程度上看，也是一张信息榜。它是在以人的财富为透镜，告诉我们最近跟经济有关的关键趋势，以及潜藏在其中的机会。</ne-text> <ne-text id="u14d7d1cf">今年，全世界一共有 3122 位企业家上榜，比去年减少了 269 人。</ne-text> <ne-text id="ubffb6da3">钟睒睒财富下降了 4%，但排名上依然是中国第一。海天味业的庞康过去一年财富下降了 17%。腾讯的马化腾下降了 25%。美的的何享健下降了 25%。字节跳动的张一鸣下降了 31%。宁德时代的曾毓群财富下降了 34%。拼多多的黄峥是为数不多财富上涨的，而且涨幅高达 63%。</ne-text> <ne-text id="ud786a25a">从领域上看，疫情尾声，奢侈品的复原能力最强。过去一年，不仅阿诺特成为世界首富，而且爱马仕、欧莱雅、香奈儿、古驰的掌门人，财富都在世界前 50。为什么奢侈品这么猛？咱们以前说过，这是因为疫情结束之后，最先开始恢复消费信心的，是那些倾向享受性消费的高消费人群，而他们最爱买的东西之一，就是奢侈品。</ne-text> <ne-text id="u33895b2d">零售、房地产、医疗健康、金融服务、软件服务、汽车及汽车零部件行业，过去一年表现都不如往年。</ne-text> <ne-text id="u7e6f0312">看哪个领域潜力大，有一个方式，就是看看有哪些新富豪上榜。那么他所在的领域，也许就是未来潜力更大的领域。</ne-text> <ne-text id="u2b38a4fc">在今年新上榜的中国富豪里，有联影医疗的薛敏，主要做的是医疗科技。还有远景能源的张雷，主要做的是风力发电。</ne-text> <ne-text id="u287f9a24">总体看，代表科创和双碳方向的企业，在未来也许有很大的发展潜力。同时，在新上榜的富豪里，也有蜜雪冰城的创始人张红超、张红甫兄弟，也成为今年的新上榜者。这说明消费品市场，假如能够找到精确的定位，并且管理好供应链，依靠规模效应也能带来巨大的收益。</ne-text>
