# 网信中国发布通知，加强“自媒体”管理

> 原文：[`www.yuque.com/for_lazy/xkrm14/iisytkmzk64kys83`](https://www.yuque.com/for_lazy/xkrm14/iisytkmzk64kys83)

正文：

网信中国发布的，关于加强“自媒体”管理的通知 1.严防假冒仿冒行为。 2.强化资质认证展示。 3.规范信息来源标注。 4.加强信息真实性管理。 5.加注虚构内容或争议信息标签。 6.完善谣言标签功能。 7.规范账号运营行为。 8.明确营利权限开通条件。 9.限制违规行为获利。 10.完善粉丝数量管理措施。 11.加大对“自媒体”所属 MCN 机构管理力度。 12.严格违规行为处置。 13.强化典型案例处置曝光。
