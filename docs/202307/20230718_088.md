# 商场外和平鸽喂养，通过产品变现，赚取利润

> 原文：[`www.yuque.com/for_lazy/xkrm14/mblme9z9t10m5aeq`](https://www.yuque.com/for_lazy/xkrm14/mblme9z9t10m5aeq)

商场外和平鸽喂养，跟喂养金鱼一个道理，商场流量大，通过以下产品变现： 1、 饲料喂养，10 元一包，15 元两包，20 元 3 包 2、 鸽子售卖，300 元/只 每月成本： 1、 场地费 5000 元， 2、 两个人力成本共 8000 元，1 人收费，1 人打扫卫生和管理鸽子。 3、购买、培训、喂养鸽子，培训鸽子不要乱飞 跟工作人员闲聊了一下，他们公司是多个地方都有在做这样的项目，湖南，江西等省份，在商场、景区等人流量大的地方开展活动，所以鸽子应该都是批量培训生产的，节约成本。
