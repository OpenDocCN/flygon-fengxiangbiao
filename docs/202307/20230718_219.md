# 烟草市场调配问题和二维码识别的痛点分析

> 原文：[`www.yuque.com/for_lazy/xkrm14/ig9kkksbmrx83d3v`](https://www.yuque.com/for_lazy/xkrm14/ig9kkksbmrx83d3v)

点赞数：78

正文：

说个半灰产的事情。 家里如果不是做类似生意的人肯定不知道，烟是不能跨地区销售的，都是地方烟草统一调配份额。 但是，有的发达地区高档烟卖的好，有的农村地区低档烟卖的好。但是烟草不会管你的需求，低档烟高档烟都是统一配比发送，所以一些地区之间的烟会通过市场自动调配的情况（私下置换，抹除地区代码）。 机会来了，以前烟草判断烟是否是哪个地区，需要通过一层类似于烫痕的代码，非常容易抹除。我刚刚跟一个烟酒店老板聊天，他说马上每一个烟上面都会有一个二维码，个体户无法识别，烟草局有一个机器扫一下就能识别是哪个地方的烟。 这其中有几个痛点 第一，同个地区，城市和农村市场对于高低端烟的品种需求失衡怎么调配？ 第二，小的烟草店已经出现了缺烟的情况，比如煊赫门这样的网红烟已经断货 5-6 天了，老板也干着急，怎么解决？ 第三，关于烟草局对二维码的识别，怎么通过技术手段屏蔽？
